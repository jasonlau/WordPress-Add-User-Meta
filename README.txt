== Description ==

This plugin simply adds custom user meta keys when a user registers an account.

== Installation ==

*Use the Plugin installer which is located in the WordPress Dashboard menu under "Plugins > Add New". OR ...
*Download add-user-meta.zip.
*Unzip add-user-meta.zip. 
*Upload the folder named 'add-user-meta' to the wp-content/plugins directory on your server
*Go to the plugins manager in your Wordpress Dashboard and locate "Add User Meta" in the list of plugins (under 'A' for 'Add User Meta').
*Click Activate.
*Go to my website and make a generous monetary donation to show your love!  

Once the plugin is activated, you will find "Add User Meta" in the Dashboard menu under "Users".
